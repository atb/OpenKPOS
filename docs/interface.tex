\chapter{Interface da Aplicação}
\label{ch:interface}

Nesta secção do manual apresentam-se e explicam-se as principais características
da interface da aplicação, isto é, da forma como a aplicação interage com o
utilizador.\newline
\indent O OpenK POS é uma aplicação vocacionada para a interação baseada em
ecrãs tácteis. Assim, a disposição dos elementos no ecrã e a sua dimensão estão
adaptados a esta situação.\newline

\begin{figure}[h!]
  \centering
      \includegraphics[width=0.8\textwidth]{./imgs_org/main_win.png}
  \caption{Interface da aplicação}
  \label{fig:main_app}
\end{figure}

\noindent A Figura ~\ref{fig:main_app} ilustra o aspecto usual da
interface da aplicação. O ecrã está dividido em três partes: uma coluna lateral esquerda na qual aparecem as
opções disponibilizadas para o perfil do utilizador; uma área maior, à direita,
na qual se apresenta o conteúdo relativo à opção actual (no caso ilustrado na
Figura ~\ref{fig:main_app} a opção activa é da \emph{Vendas}) e uma pequena área
na parte inferior do ecrã inde aparece a informação sobre a aplicação, a identificação do
utilizador actyal, o armazém actual e o nome do posto.

\section{Barra de Navegação}
Nas janelas que permitem trabalhar com vários registos é comum a utilização de
uma barra de botões com a ilustrada na Figura ~\ref{fig:nav_bar}.\newline

\begin{figure}[h!]
  \centering
      \includegraphics[width=0.8\textwidth]{./imgs_org/nav_bar.png}
  \caption{Barra de botões de navegação}
  \label{fig:nav_bar}
\end{figure}
\noindent As funcionalidades da barra (da esquerda para a direita) são as
seguintes:
\begin{itemize}
	\item Número do registo actual / Número total de registos
	\item Movimentar para o primeiro registo
	\item Movimentar para o registo anterior
	\item Carregar o registo original (pedendo os dados alterados)
	\item Movimentar para o próximo registo
	\item Movimentar para o último registo
	\item Carregar todos os registos
	\item Pesquisar registo
	\item Ordenar os registos
	\item Criar novo registo
	\item Remover o registo actual
	\item Guardar o registo actual
\end{itemize}
